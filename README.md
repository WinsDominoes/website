# Win's Personal Website
My first ever `React`, `Next.js` and `TypeScript` project!

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build
```bash
npm run build
npm run start
```
